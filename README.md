# MyPortfolio

A personal portfolio website built with **HTML, CSS, and JavaScript** to showcase my skills, projects, education, and contact details.

## 🚀 Features
- Responsive design for desktop and mobile
- About Me section with education, location, and contact info
- Skills section with progress bars
- Projects section highlighting real-world work
- Contact section with LinkedIn and GitHub links
- Downloadable resume

## 📂 Project Structure
- `index.html` → Main webpage
- `style.css` → Styling
- `script.js` → Interactivity
- `image/profile.jpg` → Profile picture
- `resume/resume.pdf` → Resume file
